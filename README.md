# Recipes App 

A full-stack web application built using Node.js, Express, MongoDB, and EJS.  
This project allows users to browse recipe-style listings, view details, and add items to favourites.  
The UI is inspired by modern food and listing platforms such as Zomato and Airbnb.

---

## Features

- Browse all recipes/listings
- View detailed recipe information
- Add and remove favourites
- Host can add, edit, and delete recipes
- Server-side rendering using EJS
- MongoDB database integration
- Clean and responsive UI using Tailwind CSS

---

**Backend**
